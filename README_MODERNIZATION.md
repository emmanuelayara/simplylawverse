## 🤝 Support & Questions

### If Form Doesn't Submit
1. Check `routes.py` is updated
2. Verify form class exists in `forms.py`
3. Check CSRF token in template
4. Check browser console for errors

### If Styles Don't Apply
1. Verify CSS is in layout.html or linked
2. Check CSS specificity (use DevTools)
3. Clear browser cache (Ctrl+Shift+Delete)
4. Check file paths are correct

### If Mobile Layout Breaks
1. Check viewport meta tag
2. Verify media queries in CSS
3. Test with DevTools responsive mode
4. Check for hardcoded widths

### For Customization Help
1. Review COMPONENT_REFERENCE.md
2. Check UI_UX_MODERNIZATION.md
3. Look at existing examples in templates
4. Use CSS DevTools to inspect
