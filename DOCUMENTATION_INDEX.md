## 🎨 Design System Quick Stats

### Colors
- Primary: `#1e3a8a`, `#3b82f6`
- Backgrounds: `#ffffff`, `#f0f4ff`
- Text: `#1e3a8a`, `#4b5563`, `#2d3748`

### Typography
- Hero Title: 2.8rem, weight 800
- Section Title: 2rem, weight 800
- Body: 0.95rem, weight 500
- Font: Segoe UI

### Spacing
- Hero Padding: 4rem
- Card Gaps: 2.5rem
- Padding: 1.5rem (standard)

### Animations
- Duration: 0.3-0.8s
- Easing: ease-out, cubic-bezier
- Types: Fade, Slide, Float, Hover, Shimmer, Bounce

### Responsive Breakpoints
- XL: ≥1600px (4 columns)
- Large: 1200-1599px (3 columns)
- Tablet: 768-899px (2 columns)
- Mobile: 577-767px (1 column)
- Small: <576px (1 column, minimal)
